# 📧 Email Spam Detection using Machine Learning

## 📌 Overview

This project focuses on building a machine learning model to classify emails as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques.

---

## 🎯 Objective

* Detect spam emails accurately
* Apply NLP techniques for text processing
* Compare machine learning models

---

## 📊 Dataset

* Text-based email dataset
* Preprocessed using cleaning and tokenization

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* NLP (TF-IDF Vectorization)

---

## 🚀 Project Workflow

1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction (TF-IDF)
4. Model Training (Naive Bayes / Logistic Regression)
5. Model Evaluation

---

## 📈 Results

* Achieved high accuracy in spam classification
* Evaluated using:

  * Confusion Matrix
  * Accuracy Score

---

## 📂 Files in Repository

* `project.ipynb` → Main notebook
* `tfidf_vectorizer.pkl` → Saved vectorizer
